**优探社交购云平台**

- 项目描述： 设计并实现了一个面向高并发和社交推荐场景的电商后端系统，采用前后端分离架构，支持短信登录、商户管理、优惠券秒杀、达人探店等功能。

- 技术栈：SpringBoot, MybatisPlus, MySQL, Redis, lombok, Docker
  - 异步秒杀: 使用Redis Stream搭建消息队列，将响应时间从460ms优化至140ms，显著降低数据库压力
  - 超卖控制: 使用Lua脚本+乐观锁机制，确保一人一单并有效防止库存超卖
  - 缓存设计: 使用 Redis 对高频访问的信息进行缓存，降低了数据库查询的压力,解决了缓存穿透、雪崩、击穿问题
  - 安全认证: 利用自定义拦截器，实现集群环境下的登录校验和权限动态刷新
  - 项目部署: Nginx实现反向代理和负载均衡
